T21 Heroic Monocible Combinaisons Simulations - Retribution

Information:
Ces simulations sont toutes basées sur les profils par défaut du T21 Heroic mythic de SimulationCraft. Set pieces and legendaries have been modified accordingly.
De plus, afin d'éviter le surplus de combinaisons, seuls les légendaires DPS ont été utilisés.
Le target error est de 0.4% ce qui signifie you can consider everything within that DPS range to be mostly equal and requiring a more detailed investigation.

Le but de ces simulations est d'avoir une idée générale des différentes combinaisons de ce tier en les comparant les unes aux autres, ce n'est pas forcément les "best builds". Plusieurs variables (comme de différents trinkets, WF/TF ou des situations en jeu) ne sont pas prises en compte. C'est pourquoi, comme toujours, vous devez simuler votre propre personnage pour trouver votre configuration optimale.
