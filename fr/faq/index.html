Foire aux Questions

Général

  • Quel est le but de ce site ?

    Nous fournissons une base centralisée des résultats de Theorycrafting et SimulationCraft pour toutes les classes de World of Warcraft.
    Notre objectif est de rendre accessible, sur un site clair et simple, toutes les informations dont vous avez besoin pour optimiser votre personnage.

  • A qui s'adresse-t-il ?

    Tous ceux qui veulent en savoir plus sur l'optimisation de leurs personnages. Avec ces résultats, nous voulons donner un aperçu de la différence entre chaque optimisation dans différentes situations, ce que vous pouvez faire pour améliorer votre équipement et quelles options peuvent vous intéresser.

  • Comment ces résultats sont-ils générés ?

    Ces résultats sont générés en utilisant le logiciel SimulationCraft, notre Ruby scripts, et plusieurs profils de simulation pour obtenir une vue d'ensemble sur plusieurs aspects d'une spécialisation.

  • Où puis-je trouver les profils SimC utilisés sur le site ?

    Nous utilisons presque toujours les profils par défaut de SimulationCraft avec nos scripts qui changent les combinaisons pour chaque configuration. Si vous êtes intéressé par les détails, vous pouvez consulter nos profils de base ici.

Combinaisons

  • Comment fonctionnent les combinaisons ?

    Pour les talents, vous pouvez spécifier un modèle pour la chaîne de talents avec les nombres 1-3 pour le talent gauche / milieu / droit de chaque niveau et un x (ou *) comme caractère générique. Par exemple, si vous entrez «xx1xx3x», seules les configurations avec le premier talent de la troisième rangée et le troisième talent de la sixième rangée s'affichent.
    Les tiers et les filtres légendaires vous permettent de sélectionner uniquement les équipements dont vous disposez pour exclure les résultats qui ne vous intéressent pas.

  • De quelle difficulté de Raid sont basées les équipements des combinaisons?

    Sauf spécification contraire, nos profils utilisent l'équipement Mythic. Les profils héroïques sont spécifiés dans la description.

  • Comment puis-je voir l'équipement d'une simulation de combinaison spécifique?

    Il n'est pas encore possible de voir les détails mais nous y travaillons!

  • Quels sont les traits T2 du Creuset de l'Halo-du-Néant utilisés par les simulations de l'anneau du Creuset?

    Les profils de chaque spécialisation utilisent toujours les trois meilleurs trait T2 de la spécialisation. Cela signifie que l'anneau du Creuset aura probablement le meilleur résultat et si vous avez des traits T2 différents, vous devriez lancer une simulation pour votre propre personnage.

  • Puis-je comparer mon personnage aux simulations des combinaisons?

    Votre personnage a des équipements différents de celles du profil Combinator (Mythic +, de guerre / forgé par les titans, etc., ce qui peut entraîner des différences très visibles). Habituellement, nous ne simulons que des équipements sans proc pour chaque niveau et difficulté. Il est fortement conseillé de simuler votre propre personnage afin de trouver la meilleure configuration d'équipements possible.

  • Mais si je dois simuler mon personnage dans tous les cas, à quoi servent les combinaisons?

    L'objectif principal de ces simulations est de trouver et de mettre en évidence certaines combinaisons d'équipements idéales. En fonction des combinaisons, vous pouvez affiner les simulations que vous souhaitez approfondir pour votre propre personnage. Même si les combinaisons utilisent des équipements différents des vôtres, la meilleure combinaison vous met généralement sur la bonne voie !

  • Est-ce que les combinaisons représentent les conditions réelles du raid?

    Pas vraiment, en ce moment. Si cela n'est pas spécifié, nous avons généralement des simulations de boss monocible sans autres joueurs ou mécaniques de boss(mouvements, adds, etc...).
    En outre, la phase 100% -90% ainsi que la phase d'exécution du boss peuvent être plus courtes ou plus longues dans une simulation, ce qui rend certains légendaires comme Cinidaria ou des talents qui fonctionnent bien à certains seuils de point de vie. Il n'y a pas de solution idéale à cela. Cela dépend fortement de votre groupe.

Reliques

  • Comment lire le graphique des reliques?

    Si vous comparez un trait au niveau de l'objet d'arme sur la première ligne, vous pouvez voir combien de niveaux d'objet le trait vaut. En survolant, vous pouvez vérifier l'augmentation DPS exacte.

Bijoux

  • Comment les simulations de bijoux sont-ils générés?

    Un seul bijou est simulé à plusieurs niveaux d'objet pour le profil de base donné. Par conséquent, l'impact de deux bijoux forts qui se synergisent très bien ne pas être pris en compte. Nous voulons ajouter une option pour cela dans le futur.

  • Le proc en raid des bijoux du Panthéon est-il inclus?

    Si un bijou du Pantheon a un suffixe "X-Pantheon", cela signifie que la simulation a supposé X autres joueurs avec le bijou dans votre raid. Sans ce suffixe, seul le proc de base a été considéré.
